# @junyi5910/agentlayer-core

Core scanner, extractor, evaluator, CI comparison, and artifact generation library for AgentLayer.

AgentLayer is a deterministic toolkit for checking whether public websites can be read, trusted, and
operated by AI agents.

## Status

`@junyi5910/agentlayer-core` is an alpha package. Generated artifacts and standards-related metadata
are review drafts, not compliance guarantees.

## Install

```bash
pnpm add @junyi5910/agentlayer-core@alpha
```

Most users should start with the CLI package instead:

```bash
pnpm dlx @junyi5910/agentlayer-cli generate https://your-site.com --out ./agentlayer-output --max-pages 20
```
